# 🎮 MediaTracker

Умный календарь релизов игр, сериалов и аниме с автоматическим поиском дат через API и самообучающейся
базой данных. Работает как десктопное приложение на Windows (Python/Tkinter).

**Разработано**  
Автор: Ukiteroa · Ukiteroa@Gmail.com · @AlukardGame

---

## Возможности

### 📰 Новости

- Сбор свежих новостей из 15+ источников
(StopGame, DTF, Игромания, Kanobu, VGTimes, IGN, GameSpot, Anime News Network, Кинопоиск)
- Фильтрация по категориям: Игры / Сериалы / Аниме / Все
- Поиск по заголовкам новостей
- Поиск по ключевому слову через Google News + RSS-источники
- Предпросмотр новости с открытием ссылки в браузере

### 🎮 Игры

- Добавление игр с платформой (PC, PS5, Xbox, Switch, Mobile)
 и статусом (Играю, В планах, Заброшено, Пройдено)
- Автоматический поиск даты выхода через **RAWG API**
- Ручное обновление даты через API
- Учёт часов, проведённых в игре
- Оценка от 0 до 10

### 🎬 Сериалы

- Добавление сериалов с текущим сезоном
- Автоматический поиск даты премьеры через **TVMaze API**
- Отслеживание просмотренных серий
- Статусы: Смотрю, В планах, Заброшено, Просмотрено

### 🎌 Аниме

- Добавление аниме с типом (TV, Movie, OVA, Special)
- Автоматический поиск даты выхода через **Jikan API**
- Отслеживание просмотренных серий
- Статусы: Смотрю, В планах, Заброшено, Просмотрено

### 📚 Манга

- Добавление манги с указанием общего количества томов
- Отслеживание прочитанных томов
- Поиск манги через Яндекс (правая кнопка мыши → контекстное меню)
- Статусы: Читаю, В планах, Заброшено, Прочитано

### 📅 Умный календарь

- Автоматическое заполнение датами релизов из добавленных элементов
- Фильтрация по типу медиа: Игры / Сериалы / Аниме / Все
- Отображаются только элементы со статусом «В планах», «Играю» или «Смотрю»
- Двойной клик по элементу календаря → переход к карточке

### 💾 Самообучающаяся база данных

- Подтверждённые пользователем даты сохраняются в локальную БД
- При повторном добавлении тайтла дата берётся из БД (мгновенно, без API)
- Статистика использования БД: кэш-хиты, вызовы API, ошибки, подтверждения пользователя

### 📊 Статистика

- Общее количество тайтлов по категориям
- Суммарное количество часов в играх
- Всего просмотренных серий/эпизодов
- Прочитано томов манги
- Количество предстоящих релизов
- Статистика самообучающейся БД

### 🎨 Интерфейс

- Тёмная тема (фиолетово-тёмный фон, неоновая подсветка)
- Фоновые изображения для каждой вкладки (опционально)
- Вкладки с переключением
- Таблицы с данными и прокруткой

### 💾 Хранение данных

- Все данные хранятся в JSON-файлах в папке `media_data/`
- Автосохранение при каждом изменении
- Устойчивость к повреждённым JSON (автоматическое создание бэкапа)

## Установка и запуск

### Требования

- Python 3.8+
- pip (менеджер пакетов Python)
- Интернет для работы API и RSS-лент

### Установка зависимостей

```bash
pip install requests feedparser pillow
```

### Запуск

```bash
python media_tracker.py
```

### Сборка в EXE (для Windows без Python)

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "MediaTracker" media_tracker.py
```

Готовый EXE появится в папке `dist/`.
