# 🎮 Game News Aggregator

Кроссплатформенное приложение для сбора и просмотра новостей игровой индустрии.

## Особенности

- 📰 Сбор новостей из множества источников (IGN, GameSpot, PC Gamer, Kotaku и др.)
- 🔍 Поиск по новостям
- ⭐ Избранные новости
- 🎯 Фильтрация по источникам
- 📅 Сортировка по дате, источнику и заголовку
- 🌙 Темная тема оформления
- 🖥️ Кроссплатформенность (Windows, macOS, Linux)
